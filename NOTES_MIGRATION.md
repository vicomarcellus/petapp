# Миграция: Добавление заметок с тегами

## Что нужно сделать

Выполни SQL-скрипты в Supabase SQL Editor в следующем порядке:

1. Открой Supabase Dashboard
2. Перейди в SQL Editor
3. Выполни содержимое файла `sql/create_notes_table.sql`
4. Выполни содержимое файла `sql/create_note_tags_table.sql`

## Что добавлено

### База данных
- Таблица `notes` для хранения заметок
- Таблица `note_tags` для тегов
- Таблица `note_tag_relations` для связи заметок и тегов (many-to-many)
- RLS политики для безопасности
- Индексы для оптимизации

### Функционал
- ✅ Создание заметок с заголовком и текстом
- ✅ Редактирование заметок
- ✅ Удаление заметок
- ✅ Закрепление важных заметок (pin)
- ✅ Система тегов (создание, удаление, выбор цвета)
- ✅ Множественные теги для одной заметки
- ✅ Поиск по заметкам и тегам
- ✅ Сортировка (закрепленные сверху, потом по дате обновления)

### UI компоненты
- Новый компонент `Input` и `Textarea` в `src/components/ui/Input.tsx`
- Стиль: bg-gray-50 с focus:bg-white и rounded-2xl
- Можно использовать во всех формах приложения

### UI
- Новая кнопка "Заметки" в навигации
- Grid-layout с карточками заметок
- Модальное окно для создания/редактирования
- Поиск в реальном времени
- Цветные теги для категоризации
- Отображение даты и времени создания/обновления

## Использование

1. Нажми "Заметки" в навигации
2. Создай новую заметку кнопкой "Добавить"
3. Заполни заголовок и текст
4. Создай теги или выбери существующие
5. При необходимости закрепи заметку
6. Используй поиск для быстрого нахождения нужной заметки
