# AI Ассистент для трекера здоровья кота

## Настройка

1. Получите API ключ от OpenAI: https://platform.openai.com/api-keys

2. Создайте файл `.env` в корне проекта:
```
VITE_OPENAI_API_KEY=sk-your-api-key-here
```

3. Перезапустите dev сервер:
```bash
npm run dev
```

## Использование

1. Откройте форму добавления записи
2. Нажмите кнопку "AI Помощник" (иконка Sparkles)
3. Напишите в свободной форме о состоянии кота

### Примеры запросов:

- "Сегодня состояние хорошее, дал преднизолон 0,3 в 10:00"
- "Плохо, была рвота утром, не ест"
- "Отлично себя чувствует, активный, играет"
- "Средне, дрожь была, дал лекарство"
- "Критично, лежит, не реагирует"

AI автоматически:
- Определит оценку состояния (0-4)
- Извлечет симптомы (рвота, дрожь, боль, ел ли сам)
- Выделит информацию о лекарствах
- Создаст заметку

## Модель

Используется `gpt-4o-mini` - быстрая и недорогая модель от OpenAI.

Стоимость: ~$0.15 за 1M входных токенов, ~$0.60 за 1M выходных токенов.

Один запрос ≈ 500 токенов ≈ $0.0003 (0.03 цента).
