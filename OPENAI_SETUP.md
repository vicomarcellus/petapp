# Настройка OpenAI для AI помощника

## Что это

AI помощник - это умный чат-бот, который **выполняет команды** и управляет данными о питомце.

## Что он умеет

### Добавление записей
- `состояние 4` - добавит запись состояния с оценкой 4/5
- `дрожь` - добавит симптом "Дрожь"
- `дали преднизолон 0,3` - добавит лекарство
- `утром было 2, сейчас 4` - добавит несколько записей состояния
- `состояние 3 в 9 утра` - добавит с указанием времени

### Удаление
- `удали дрожь` - удалит симптом
- `удали состояние в 10 утра` - удалит конкретную запись
- `удали все симптомы` - очистит все симптомы за день

### Вопросы
- `как дела?` - покажет сводку за сегодня
- `что было вчера?` - покажет данные за вчера

## Настройка на Vercel

1. Зайдите в настройки проекта: https://vercel.com/your-project/settings/environment-variables

2. Добавьте переменную:
   - **Name**: `VITE_OPENAI_API_KEY`
   - **Value**: ваш OpenAI ключ (из файла `.env`)
   - **Environment**: Production, Preview, Development (все)

3. Нажмите **Save**

4. Редеплой:
   - Deployments → три точки → Redeploy

## Как использовать

1. Откройте календарь
2. Нажмите фиолетовую кнопку со звёздочкой (правый нижний угол)
3. Пишите команды естественным языком:
   - "состояние 4"
   - "дрожь"
   - "дали преднизолон 0,3 в 17:00"
   - "удали рвоту"
   - "как дела?"

AI понимает русский язык, склонения, синонимы и разный порядок слов.

## Технические детали

- Использует GPT-4o-mini для парсинга команд
- Работает с Supabase для сохранения данных
- Видит контекст текущего дня (существующие записи)
- Автоматически определяет время ("сейчас", "утром", "в 17:00")
